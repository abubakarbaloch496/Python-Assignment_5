# Python-Assignment_5
Assignment: Simple Python Program 
Task Description
1. Ask the user to enter their name.
2. Ask the user to enter a number.
3. Check whether the number is even or odd.
4. Print numbers from 1 to the given number using a for loop.
5. Use a while loop to print numbers from the given number down to 1.
 6. Display a thank you message using the user's name. 
Example Output
Enter your name: Ali
Enter a number: 5 
 
5 is Odd 
 
Numbers from 1 to 5:
1 2 3 4 5 
 
Countdown:
5 4 3 2 1 
 
Thank you Ali!
